# 🗺️ Spatial Social Network - شبكة التواصل الاجتماعي المكانية

منصة تواصل اجتماعي حديثة تجمع بين التواصل الاجتماعي والذكاء الجغرافي، مع خريطة تفاعلية كواجهة رئيسية بدلاً من Timeline التقليدي.

## 🎯 المميزات الرئيسية

### 🔐 نظام المصادقة
- تسجيل المستخدمين الجدد مع التحقق
- تسجيل الدخول الآمن باستخدام JWT
- إدارة الجلسات والتوكنات

### 🗺️ الخريطة التفاعلية
- عرض المنشورات كنقاط مكانية على خريطة Mapbox
- تكبير/تصغير والتنقل السلس
- مجموعات المنشورات (Clustering) للأماكن المزدحمة
- لا أدوات GIS معقدة - تجربة مستخدم بسيطة

### 📍 المنشورات المكانية
- إنشاء منشور جديد بالموقع الحالي
- التقاط صور مباشرة من الكاميرا
- رفع الصور الموجودة
- إضافة وصف نصي للمنشور
- تخزين تلقائي للإحداثيات والزمن

### 👥 نظام الأصدقاء
- البحث عن المستخدمين بالاسم
- إرسال طلبات الصداقة
- قبول/رفض الطلبات الواردة
- عرض قائمة الأصدقاء
- رؤية منشورات الأصدقاء فقط

### 💬 الدردشة الفورية
- محادثات فورية بين الأصدقاء
- WebSocket للرسائل الفورية
- حالة الاتصال (متصل/غير متصل)
- إشعارات الرسائل الجديدة

### 📊 التحليل المكاني (Future)
- كثافة المنشورات في المناطق
- أكثر الأماكن نشاطاً
- التغير الزمني للمحتوى
- تحليلات جغرافية متقدمة

## 🛠️ التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة واجهة المستخدم
- **Mapbox GL JS** - خرائط تفاعلية
- **Socket.IO Client** - اتصال WebSocket
- **Axios** - طلبات HTTP
- **CSS Modules** - تصميم معزول

### Backend
- **Node.js & Express** - خادم API
- **PostgreSQL 14+** - قاعدة البيانات
- **PostGIS** - إضافة جغرافية لـ PostgreSQL
- **Socket.IO** - WebSocket للدردشة
- **JWT** - مصادقة آمنة
- **Bcrypt** - تشفير كلمات المرور

### DevOps & Tools
- **npm** - إدارة الحزم
- **dotenv** - متغيرات البيئة
- **Vite** - أداة البناء السريعة

## 🚀 البدء السريع

راجع ملف `.agent/workflows/project-setup.md` للحصول على دليل مفصل خطوة بخطوة.

### تثبيت سريع
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (في نافذة أخرى)
cd client
npm install
npm run dev
```
